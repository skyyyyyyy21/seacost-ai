# 标准化验收入库 SOP

## 适用 Agent
- inventory

## 关键词
验收, 入库, 称重, 拍照, 偏差, 退货, 盘点, 临期, 保质期

## 描述
海鲜酒楼标准化验收入库和库存管理流程，覆盖从食材到货到库存管理的全链路。

## 规范内容

### 一、验收入库流程（3步闭环）

#### 1. 称重验收
- 海鲜专用防水电子秤直连系统
- 称重数据自动同步，无需手工录入
- 重量偏差超过 ±5% 自动触发总部审批

#### 2. 质量检查
- 验收时拍照留证（外观、鲜活度）
- 记录产地、批次、到货时间
- 鲜活度评级：A（鲜活）、B（一般）、C（临死）、D（死亡）

#### 3. 入库操作
- 系统自动分配存储位置（海鲜区、冷冻区、干货区）
- 打印标签（品名、批次、入库日期、保质期）
- 不合格品一键发起退货，流程留痕可追溯

### 二、库存管理（精细化）

#### 1. 分批次管理
- 系统强制先进先出（FIFO）
- 同品名不同批次分开存放
- 出库时自动匹配最早批次

#### 2. 保质期预警
- 临期前 3 天自动推送"优先使用"提醒
- 临期前 1 天推送"紧急处理"提醒
- 过期食材自动锁定，禁止出库

#### 3. 盘点流程
- 支持分档口快速盘点（海鲜区、火锅区、后厨）
- 系统自动生成盘点差异表
- 差异超过 5% 需复盘确认

#### 4. 损耗管理
- 报损、调拨流程化
- 每一笔损耗必须记录原因（自然损耗、操作失误、质量问题）
- 月度损耗率目标：海鲜 ≤ 3%，其他食材 ≤ 1%

### 三、关键控制点
1. **验收双人制**：验收必须两人同时在场（厨师长 + 仓管）
2. **拍照留证**：每批次验收必须拍照，保存 30 天
3. **温度监控**：冷链食材到货温度必须 ≤ 4°C

### 四、禁止事项
- 禁止未验收直接入库
- 禁止手工修改称重数据
- 禁止过期食材继续使用
