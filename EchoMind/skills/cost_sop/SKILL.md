# 自动成本核算 SOP

## 适用 Agent
- cost

## 关键词
成本, 毛利, 核算, 利润, 报损, 成本分析, 毛利分析

## 描述
海鲜酒楼自动成本核算标准作业流程，覆盖从采购成本到菜品毛利的精准核算。

## 规范内容

### 一、成本核算流程（自动闭环）

#### 1. 采购成本归集
- 系统自动抓取每笔采购的单价、数量、供应商
- 按食材分类归集（海鲜、肉类、蔬菜、调料、酒水）
- 支持按批次核算（同品名不同批次价格可能不同）

#### 2. 加工成本分摊
- 海鲜加工损耗率自动计算（如龙虾去壳损耗 30%）
- 调料成本按菜品标准配方分摊
- 人工成本按档口分摊（可选）

#### 3. 菜品成本计算
- 每道菜品的标准成本 = 主料 + 辅料 + 调料 + 加工损耗
- 实际成本 = 标准成本 × (1 + 损耗率偏差)
- 支持按日/按周/按月汇总

#### 4. 毛利分析
- 菜品毛利 = 售价 - 实际成本
- 毛利率 = 毛利 / 售价 × 100%
- 目标毛利率：海鲜类 60-70%，其他菜品 55-65%

### 二、成本异常分析

#### 1. 异常检测
- 单菜品成本偏差超过 10% 自动预警
- 品类毛利率低于目标 5% 以上触发分析
- 损耗率异常（超过历史均值 2 倍）自动标记

#### 2. 异常定位
- 系统自动定位异常来源：
  - 采购价上涨？
  - 加工损耗增加？
  - 售价调整未同步？
  - 配方变更未更新？

#### 3. 优化建议
- 根据异常类型自动生成优化建议：
  - 采购价上涨 → 建议更换供应商或调整售价
  - 损耗增加 → 建议检查加工流程或培训厨师
  - 毛利率偏低 → 建议优化菜品结构或调整定价

### 三、报表体系

#### 1. 日报
- 今日采购总额、各品类占比
- 今日成本异常清单
- 今日毛利率概览

#### 2. 周报
- 本周成本趋势分析
- 本周损耗率统计
- 本周毛利 TOP10 / BOTTOM10 菜品

#### 3. 月报
- 月度成本核算汇总
- 月度毛利分析（按品类、按档口）
- 月度损耗分析（按原因分类）
- 成本优化建议

### 四、关键控制点
1. **配方标准化**：每道菜品必须有标准配方卡（BOM）
2. **售价同步**：售价调整必须在 24 小时内同步到系统
3. **成本复核**：每月财务复核成本核算结果

### 五、禁止事项
- 禁止手工修改成本数据（必须通过流程调整）
- 禁止跳过配方直接估算成本
- 禁止隐瞒损耗（必须如实记录）
